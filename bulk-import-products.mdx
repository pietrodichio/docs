## Overview

The bulk import endpoint allows you to add multiple products to your catalog in a single request. This is useful for initial catalog setup or synchronizing products from your e-commerce platform.

## Authentication

This endpoint requires a valid JWT token in the Authorization header:
```bash
Authorization: Bearer your_jwt_token
```

## Rate Limiting

The bulk import endpoint is limited to 1000 products per request to ensure optimal performance.

## Best Practices

1. **Validate Data**: Ensure all required fields are present and properly formatted before sending.
2. **Batch Size**: Keep batches under 1000 products to avoid rate limits.
3. **Error Handling**: Implement retry logic with exponential backoff for rate limit errors.
4. **Idempotency**: The operation is idempotent - sending the same product data multiple times won't create duplicates.
